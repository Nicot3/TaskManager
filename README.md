# TaskManager
An easy C# project to manage tasks
## Functions:
- Create tasks with name, description, tags, start date, end date
- Delete tasks
- Update tasks
- Mark tasks as completed

## Practices
- Clean code
- Domain Driven Design (Aplication/Infrastructure/Domain layers)
- CQRS
- Repository Pattern
- SOLID
- Unit Testing

## Technologies
- ASP.NET Core 8
- Entity Framework Core
- AutoMapper
- Blazor ServerRender
